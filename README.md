# N8N工作流仓库

此仓库存储我在YouTube频道分享的N8N工作流配置文件，用户可直接下载JSON文件导入N8N使用。

### 个人信息
1. YouTube频道：https://www.youtube.com/@linbintalk
2. AI学长的博客：https://ai.linbintalk.com/

## 使用指南
1. 访问[workflows目录](./workflows)
2. 下载所需JSON配置文件
3. 在N8N中点击"Import from file"导入
4. 根据您的需求调整参数

## 贡献指南
欢迎提交您的工作流配置！请确保：
- 移除敏感信息（API密钥等）
- 包含简要说明文档
- 测试工作流功能正常
